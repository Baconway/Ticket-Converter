# Generate an image from your .pdf Ticket to use with Google Wallet (or any other wallet app)

And never have to use online converters ever again!

## Setup

1. Create 2 Folders `results` and `tickets`
2. Put your .pdf ticket (1 page) into the `tickets` folder
3. Run
   `node converter.js`
4. Check `results` to get your outputted ticket (same name as your .pdf)
